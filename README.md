## Course Overview
This course teaches Reinforcement Learning and Deep Learning techniques to create practical AI programs using TensorFlow and OpenAI Gym, focusing on building intelligent agents for real-world problems.

## What You'll Learn
- Artificial Neural Networks with TensorFlow
- Convolutional Neural Networks (CNNs) for image processing
- Tabular Q-Learning and SARSA
- Deep Q-Learning (DQN) with CNNs
- Cross Entropy Method for RL
- Double DQN and Dueling DQN
- Custom OpenAI Gym environments
